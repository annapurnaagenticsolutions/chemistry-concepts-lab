## Authentication status legend

- **Concept-reviewed:** The presented concept is aligned with standard school-level treatment and was checked for obvious factual errors, unsafe wording, and overclaiming.
- **Simplified with caveat:** The idea is simplified for learners and includes guardrails so the simplification does not become false.
- **Needs future chapter mapping:** The concept is acceptable for a demo/pilot, but each lesson should later be mapped to exact NCERT/CBSE chapter/page references.

## Known limits

- Not a laboratory manual.
- No unsupervised experiments with unknown chemicals, cleaners, acids, bases, heat, glass, or flames.

## Honest release statement

This package presents a curated, visual, low-risk learning aid. It supports understanding and curiosity, but formal curriculum decisions should use official NCERT/CBSE material and teacher judgement.
